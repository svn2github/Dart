Repository Software Site for the Dart Plugin for Eclipse

This software site provides the Dart Plugin for Eclipse.

To use this software site in your existing installation of Eclipse, add one of

http://www.dartlang.org/eclipse/update/channels/dev/ (latest weekly build)

http://www.dartlang.org/eclipse/update/channels/stable/ (latest stable minor version)

to your list of available software sites ("Help > Install new software...").

Do note that this current update will not pull in the latest SDK. You'll need to do that manually from the appropriate channel:

http://gsdview.appspot.com/dart-archive/channels/dev/release/latest/sdk/ (latest weekly build)

http://gsdview.appspot.com/dart-archive/channels/stable/release/latest/sdk/ (latest stable minor version)
